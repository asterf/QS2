# Quick TOTP Bookmarklet

Quick Skip 2 Factor Authentication is a lightweight script that instantly generates 2FA (TOTP) codes from your browser bookmarks.

## How to use

0. Run [this code](https://github.com/asterf/QS2/raw/main/script.js) as a bookmarklet, or execute it in your browser devTools console.  
1. Press Enter.  
2. Enter the secret seed in Base32 format.  
3. Then enter the hash algorithm (or just press Enter to skip—SHA-1 will be used, which is usually fine).  
4. After a short moment, a code should appear. Copy that code and use it for your 2FA.

## How to find the hash algorithm and secret seed

Scan the QR code shown when setting up 2FA and get the text starting with `otpauth://`.  
In that text:  
- The value starting with `algorithm=` is the hash algorithm.  
- The value starting with `secret=` is the secret seed.
